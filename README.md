## About the app
App enables users to convert various image files into file type of their choice. Both file by file and batch importing/exporting are supported. Supported formats: BMP, GIF, JPG, PNG, TIF, WEBP.

- Batch importing/exporting.
- Supported input formats:  BMP, GIF, JPG, PNG, TIF<sup>1</sup>, WEBP
- Supported output formats: BMP<sup>1</sup>, JPG, PNG, TIF<sup>1</sup>, WEBP

<sup>1</sup> - if the app is obtained via Google Play, the feature is available in paid version only.
